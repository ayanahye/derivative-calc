Instructions

Format For Calculator

To write an exponent: Use '**' rather than '^'.

To multiply variables or functions use '*' only.

To compute trigonometric equations: Make sure the angle is in brackets '()'.

Examples

1) x ** 2 + 1 // prints 2*x

2) sin(x) * cos(x) // prints -sin(x)**2 + cos(x)**2

3) 3*x**2 * cos(x) // prints -3*x**2*sin(x) + 6*x*cos(x)

4) ln(sin(x)) // prints cos(x)/sin(x)
